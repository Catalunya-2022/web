# Política de seguretat / Security Policy

## Com informar d'una vulnerabilitat / Reporting a vulnerability

Si trobes una vulnerabilitat en aquest codi o al lloc públic `2022.cat`, informa'n de manera privada mitjançant [GitHub Security Advisories](https://github.com/catalunya-2022/web/security/advisories/new). No obris issues públiques per a informes de seguretat.

If you find a vulnerability in this code or on the public site at `2022.cat`, report it privately via [GitHub Security Advisories](https://github.com/catalunya-2022/web/security/advisories/new). Please do not open public issues for security reports.

- Confirmarem la recepció en pocs dies laborables. / We will acknowledge receipt within a few working days.
- No hi ha programa de recompenses. / There is no bounty program.

## Abast / Scope

- Aquest repositori i el lloc públic a `https://2022.cat`. / This repository and the public site at `https://2022.cat`.
- El lloc és estàtic: no té comptes d'usuari, ni autenticació, i no emmagatzema dades de visitants (les analítiques són sense cookies). / The site is static: no user accounts, no authentication, and no visitor data stored (analytics are cookie-free).
- Per al servidor MCP (`mcp.2022.cat`), fes servir el [repositori mcp](https://github.com/catalunya-2022/mcp). / For the MCP server (`mcp.2022.cat`), use the [mcp repository](https://github.com/catalunya-2022/mcp).

## Versions suportades / Supported versions

Només la branca `main` i el desplegament de producció actual. / Only the `main` branch and the current production deployment.
